# OpenCode Configuration

Place your OpenCode configuration files here.

## Structure

```
opencode/
├── settings.json         # OpenCode settings
├── keybindings.json      # Custom keybindings
└── README.md
```

## Setup

1. Install OpenCode from https://opencode.ai
2. Place your config files here
3. Run the installer to symlink configs

## Config Locations

- **macOS**: `~/Library/Application Support/OpenCode`
- **Linux**: `~/.config/OpenCode`

## Tips

- Export your current OpenCode config before migrating
- Backup any extensions list you want to preserve
