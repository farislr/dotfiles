# Tech Stack

## Language
- **Python 3.8+** (primary implementation language)

## Dependencies
- **pyyaml>=6.0**: YAML profile parsing and merging
- **requests>=2.31.0**: HTTP requests for downloads
- **rich>=13.7.0**: Terminal UI (prompts, tables, progress bars)

## Package Managers
- **macOS**: Homebrew (brew)
- **Ubuntu/Debian**: APT (apt)
- **Arch/Manjaro**: Pacman (pacman)

## Configuration Format
- **YAML**: Profile definitions (profiles/*.yml)
- **Shell Scripts**: Plugin installation helpers (scripts/*.sh)

## System Integration
- **Symlinks**: Live config deployment strategy
- **Git**: Version control for dotfiles
- **Oh-My-Zsh**: Zsh framework with plugins
- **Oh-My-Posh**: Cross-platform prompt engine
